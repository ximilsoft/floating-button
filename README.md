# Floating Button
Requirements
- HTML
- CSS
- Font Awesome
- Bootstrap (Optional)

```html
<style>

  position: fixed;
  width: 60px;
  height: 60px;
  bottom: 20px;
  right: 20px;
  background-color: #168AFF;

  }

  }
</style>

</a>
```
